# fily

**Code context engine for coding agents and humans.**

fily indexes a codebase into a call-graph-aware semantic index and serves
precise code context to AI coding agents (via MCP) and to developers —
retrieval with the recall of grep and none of the noise.

## Status

🚧 **Pre-release.** The crates published from this repository are placeholders
reserving the names while the initial release is prepared.

- Website: [fily.ai](https://fily.ai)
- Crates: `fily` (CLI), `fily-core` (engine library)
